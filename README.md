# 🏨 Hotel Booking Cancellation Prediction (ML Project)

An end-to-end Machine Learning project that predicts whether a hotel booking will be cancelled based on historical booking data.

This project demonstrates the complete ML lifecycle — from data preprocessing and exploratory data analysis (EDA) to model building and evaluation.

---

## 🚀 Features

- 📥 Data loading and preprocessing
- 🧹 Data cleaning and handling missing values
- 📊 Exploratory Data Analysis (EDA)
- ⚙️ Feature engineering
- 🤖 Multiple ML models (basic → advanced)
- 🎯 Best accuracy achieved: **87%**
- 📈 Model comparison and evaluation

---

## 🧠 ML Workflow

1. Data Collection & Loading
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training:
   - Logistic Regression
   - Decision Tree
   - Random Forest
   - (and other models used)
6. Model Evaluation & Comparison
7. Final Model Selection

---

## 📊 Problem Statement

Predict whether a hotel booking will be:

- ✅ Not Cancelled
- ❌ Cancelled

---

## 📈 Results

- Best Model Accuracy: **87%**
- Evaluated using:
  - Accuracy
  - Confusion Matrix
  - Classification Report
